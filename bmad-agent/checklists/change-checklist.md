# Change Navigation Checklist

version: 1.0.0
name: Change Navigation Checklist
description: Guide for analyzing and planning significant changes during the BMAD workflow.
id: change-checklist
metadata:
  authors:
    - author1
    - author2
  tags:
    - change-management
    - navigation

checklist:

- category: Understand the Trigger & Context
  items:
  - Identify Triggering Story
  - Define the Issue
  - Assess Initial Impact
  - Gather Evidence

- category: Epic Impact Assessment
  items:
  - Analyze Current Epic
  - Analyze Future Epics
  - Summarize Epic Impact

- category: Artifact Conflict & Impact Analysis
  items:
  - Review PRD
  - Review Architecture Document
  - Review Frontend Spec (if applicable)
  - Review Other Artifacts (if applicable)
  - Summarize Artifact Impact

- category: Path Forward Evaluation
  items:
  - Option 1: Direct Adjustment / Integration
  - Option 2: Potential Rollback
  - Option 3: PRD MVP Review & Potential Re-scoping
  - Select Recommended Path

- category: Sprint Change Proposal Components
  items:
  - Identified Issue Summary
  - Epic Impact Summary
  - Artifact Adjustment Needs
  - Recommended Path Forward
  - PRD MVP Impact
  - High-Level Action Plan
  - Agent Handoff Plan

- category: Final Review & Handoff
  items:
  - Review Checklist
  - Review Sprint Change Proposal
  - User Approval
  - Confirm Next Steps
